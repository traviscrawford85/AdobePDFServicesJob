# Test Job: PDF Services Automation

This repository contains a Python script that uses Adobe PDF Services API to perform PDF automation tasks, such as exporting PDF content to other formats (e.g., Word documents). The project is designed for quick integration and testing of Adobe PDF Services API functionalities.

---

## Features

- Exports PDF content to Microsoft Word (`.docx`) format.
- Demonstrates the use of environment variables for secure API credential management.
- Integrates Adobe PDF Services API with a Python script for automation.

---

## Prerequisites

1. **Python 3.10 or higher**: Ensure Python is installed on your system.
2. **Adobe PDF Services API**: Obtain an API key and credentials from the [Adobe Developer Console](https://developer.adobe.com/document-services/).
3. **Required Python Packages**:
   - `python-dotenv`
   - `adobe-pdfservices-sdk`
